# 워크스페이스 일반 설정

워크스페이스 이름·설명을 수정하고, 리소스 할당량·계정 시트·활성 기간 등을 확인합니다.

> Runway 워크스페이스 > **설정** > **일반** 메뉴

일반 설정 페이지에는 다음 항목이 표시됩니다.

| 영역 | 표시 항목 | 워크스페이스 관리자 수정 가능 여부 |
|---|---|---|
| **일반** | 이름, ID, 설명, 생성일, 생성자 | 이름·설명만 수정 가능 |
| **리소스 할당량** | CPU, Memory, Storage, GPU | 수정 불가 (조회만 가능) |
| **계정 시트** | 최대 사용자 수 | 수정 불가 (조회만 가능) |
| **활성 기간** | 시작 날짜, 만료일 | 수정 불가 (조회만 가능) |

> **Info**: 설정 변경이 필요한 경우
>
> 워크스페이스의 리소스 할당량, 계정 시트 수, 활성 기간 등은 **플랫폼 관리자가 지정한 값**으로 유지됩니다.
> 이름·설명을 제외한 항목의 변경이 필요하면 **플랫폼 관리자에게 문의**하세요.

---

## 워크스페이스 이름·설명 수정하기

**워크스페이스 관리자**는 워크스페이스의 이름과 설명을 수정할 수 있습니다.

1.  워크스페이스에서 **설정** > **일반** 메뉴로 이동합니다.

    

2.  **일반** 영역의 **편집** 버튼을 클릭합니다.

3.  **이름**과 **설명**을 수정한 후 **저장** 버튼을 클릭합니다.

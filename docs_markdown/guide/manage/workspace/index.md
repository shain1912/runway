# 워크스페이스 관리

워크스페이스는 Runway 플랫폼에서 조직 (또는 팀) 단위로 사용자, 프로젝트, 리소스를 관리하는 공간입니다.  
워크스페이스 사용자는 역할에 따라 프로젝트를 생성하고, 사용자를 초대·관리하며, 역할과 리소스를 운영할 수 있습니다.

> Runway 워크스페이스 > 좌측 네비게이션 메뉴

---

## 주요 내용

-    **프로젝트 생성**

    ---

    워크스페이스 내에 새 프로젝트를 생성하고 리소스 쿼터를 설정합니다.

     [프로젝트 생성](project-create.md)

-    **사용자 관리**

    ---

    사용자를 초대하고, 역할을 부여·변경하며, 제거 또는 비활성화합니다.

     [워크스페이스 사용자 관리](workspace-users.md)

-    **역할 관리**

    ---

    시스템 기본 역할을 확인하고, 권한을 조합한 사용자 정의 역할을 생성·수정·삭제합니다.

     [워크스페이스 역할 관리](workspace-roles.md)

-    **리소스 모니터링**

    ---

    워크스페이스·프로젝트 단위의 리소스 사용 현황과 트렌드를 확인합니다.

     [워크스페이스 리소스 모니터링](../../operate/monitoring/workspace-monitoring.md)

-    **일반 설정**

    ---

    워크스페이스 이름·설명을 수정하고, 리소스 할당량·계정 시트·활성 기간 등을 확인합니다.

     [워크스페이스 일반 설정](workspace-info.md)

---

## 역할별 주요 작업

| 작업 | 메뉴 위치 | 수행 가능한 역할 | 설명 |
|---|---|---|---|
| 프로젝트 생성 | 프로젝트 | 관리자, **멤버** | 워크스페이스 내 새 프로젝트를 생성합니다. |
| 사용자 초대 | 멤버 | 관리자 | 이메일로 워크스페이스에 사용자를 초대합니다. |
| 사용자 관리 | 멤버 | 관리자 | 사용자의 역할 부여·변경, 비활성화·제거를 수행합니다. |
| 역할 생성·관리 | 설정 > 역할 | 관리자 | 권한을 조합한 사용자 정의 역할을 생성·수정·삭제합니다. |
| 리소스 모니터링 | 모니터링 | 관리자 | 워크스페이스 전체 및 프로젝트별 리소스 사용 현황을 조회합니다. |
| 일반 설정 변경 | 설정 > 일반 | 관리자 | 워크스페이스 이름·설명을 수정합니다. |

> **Info**: 사용자 정의 역할
> 위 표는 시스템 기본 역할(관리자, 멤버, 뷰어) 기준입니다.
> **사용자 정의 역할**을 통해 관리자가 아닌 사용자에게도 개별 권한을 부여할 수 있습니다.
> 자세한 내용은  **[워크스페이스 역할 관리](workspace-roles.md)**를 참고하세요.

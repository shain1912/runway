# 내 계정 및 키 관리

계정 설정에서는 개인 프로필 정보 수정, 비밀번호 변경, 액세스 키 관리, Kubeconfig 활용, 언어 설정 등 사용자 계정과 관련된 작업을 수행할 수 있습니다.

> 오른쪽 상단 프로필 아이콘

---

## 주요 내용

-    **계정/언어 설정**

    ---

    프로필 정보를 수정하고, 비밀번호를 변경하며, 역할을 확인하고, 서비스 언어를 변경합니다.

     [계정/언어 설정](account-profile.md)

-    **액세스 키 관리**

    ---

    API 키, LLM API 키, S3 키를 생성하고 관리합니다.

     [액세스 키 관리](access-keys.md)

-    **Kubeconfig 활용**

    ---

    Kubernetes 클러스터 접근을 위한 인증 설정 파일을 다운로드하고 사용합니다.

     [Kubeconfig 활용](kubeconfig.md)

> **Info**: Runway 서비스 및 관리센터 공통 적용
>
> 계정 관리 메뉴는 **Runway 서비스**와 **Runway 관리센터**에 모두 동일하게 적용됩니다.

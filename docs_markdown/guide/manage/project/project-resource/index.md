# 프로젝트 리소스 관리

프로젝트별로 설정된 컴퓨팅 리소스의 최대 사용 한도(할당량)는 필요에 따라 조정할 수 있습니다.

> Runway 워크스페이스 > 프로젝트 > **설정** > **일반** 메뉴

> **Note**: 프로젝트 리소스 변경 권한
>
> 프로젝트 리소스 할당량은 **프로젝트 관리자(Admin)**만 변경할 수 있습니다.

## 프로젝트 리소스 할당량 변경하기

1.  프로젝트에서 **설정** > **일반** 메뉴로 이동합니다.

    

2.  **리소스 할당량** 영역 오른쪽의 **편집** 버튼을 클릭합니다.

3.  각 리소스 항목에 대한 할당량을 조정합니다.

    
    
    > **Info**: 최대 자원량 확인
    >
    > 각 입력 필드 오른쪽에는 현재 이 프로젝트에서 **설정 가능한 최대 자원량**이 표시됩니다.  
    > 이 값은 워크스페이스에 할당된 자원 한도와 현재 사용 상황을 기준으로 자동 계산됩니다.
    >
    > - **CPU**: 프로젝트에서 사용할 수 있는 최대 코어 수
    > - **메모리**: 프로젝트에서 사용할 수 있는 최대 메모리 용량
    > - **디스크**: 프로젝트에서 사용할 수 있는 최대 디스크 용량
    > - **GPU**: 프로젝트에서 사용할 수 있는 GPU 수  
    >   (워크스페이스에 할당된 노드에 따라 자동으로 표시됩니다. GPU가 할당되지 않은 경우 0으로 표시됩니다.)

4.  변경 내용을 입력한 후 **저장** 버튼을 클릭합니다.

# 프로젝트 삭제

프로젝트 삭제는 되돌릴 수 없는 작업입니다. 삭제 전 반드시 영향 범위를 확인하세요.

> Runway 워크스페이스 > 프로젝트 > **설정** > **일반** 메뉴

> **Note**: 프로젝트 삭제 권한
>
> 프로젝트 삭제는 **프로젝트 관리자(Admin)**만 수행할 수 있습니다.

---

## 프로젝트 삭제하기

> **Warning**: 삭제 전 확인사항
>
> 프로젝트를 삭제하면 다음 항목이 **영구적으로 제거**됩니다:
>
> - 프로젝트에 포함된 **애플리케이션, 볼륨, 파이프라인**
> - 등록된 **모델 및 배포된 서비스(엔드포인트)**
> - 프로젝트 **설정 및 관련 리소스 구성**
>
> 삭제 전에 필요한 데이터와 모델을 반드시 백업하세요.

1.  프로젝트 페이지에서 **설정** > **일반** 메뉴로 이동합니다.

    

2.  화면 하단의 **프로젝트 삭제** 버튼을 클릭합니다.

3.  확인 팝업에서 **프로젝트 이름**을 정확히 입력합니다.

    
    
    > **Info**: 삭제 확인 문자열
    >
    > 실수로 인한 삭제를 방지하기 위해 프로젝트 이름을 정확히 입력해야 합니다. 오탈자가 있으면 삭제가 진행되지 않습니다.

4.  **삭제** 버튼을 클릭합니다.

# 프로젝트 이름 변경

프로젝트의 기본 정보(이름, 설명)를 확인하고 수정할 수 있습니다.

> Runway 워크스페이스 > 프로젝트 > **설정** > **일반** 메뉴

> **Note**: 프로젝트 정보 수정 권한
>
> 프로젝트 기본 정보는 **프로젝트 관리자(Admin)**만 수정할 수 있습니다.

## 프로젝트 일반 정보 수정하기

프로젝트의 이름을 변경할 수 있습니다.

> **Warning**: 변경한 프로젝트 이름 공유
>
> 프로젝트 이름 변경은 프로젝트 멤버에게 혼란을 줄 수 있으므로 변경 전/후 프로젝트 관련 사용자에게 공유하는 것을 권장합니다.

1.  프로젝트 페이지에서 **설정** > **일반** 메뉴를 선택합니다.

    

2.  **일반** 영역의 **편집** 버튼을 클릭합니다.

3.  프로젝트 **이름**을 수정하고, 필요하면 **설명(선택 사항)** 내용을 수정합니다.

    

4.  **저장** 버튼을 클릭하여 변경사항을 반영합니다.

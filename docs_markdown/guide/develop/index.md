# 개발하기

Runway는 여러 사용자가 공통 인프라 위에서 각자의 개발 방식과 도구를 자유롭게 사용할 수 있도록 **표준화된 실행 환경과 자원 관리 체계**를 제공합니다.
이를 위해, AI·ML 개발의 모든 작업을 **애플리케이션 단위로 실행**하는 구조를 기반으로 합니다.

개발 도구(JupyterLab, Code Server), 데이터 저장소, 학습 파이프라인까지 — 다양한 구성 요소를 프로젝트 내에서 **애플리케이션으로 배포되고 관리**할 수 있습니다.
이 섹션에서는 다양한 AI·ML 개발을 수행할 수 있도록 필요한 실행 환경을 구성하고, 자원을 할당하며, 애플리케이션을 배포·관리하는 방법을 설명합니다.

| 🛠️ 개발 환경 구성 | | 🚀 서비스 제공 |
|:---:|:---:|:---:|
| Code ServerJupyterLab벡터 DB | **→**모델 개발 | 파이프라인대시보드 |
| *앱 카탈로그 배포* | | *Helm 직접 배포* |

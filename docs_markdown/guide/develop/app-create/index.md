# 애플리케이션 생성

AI·ML 개발을 수행하려면 코드를 작성하고 데이터를 다룰 수 있는 실행 환경이 필요합니다.

Runway에서는 **플랫폼 앱**, **앱 카탈로그**, **Helm 직접 배포** 세 가지 방식으로
프로젝트에 필요한 개발 도구와 인프라를 애플리케이션 형태로 배포할 수 있습니다.

## 애플리케이션 배포 방식

| 배포 방식 | 대상 | 특징 |
|----------|------|------|
| **플랫폼 앱** | Gitea, MLflow, Airflow 등 | 워크스페이스 단위 자동 제공, 즉시 사용 |
| **앱 카탈로그** | JupyterLab, Code Server, 벡터 DB 등 | 간소화된 설정으로 빠른 배포 |
| **직접 배포** | 모든 Helm 차트 | values.yaml 직접 구성, 완전한 제어 |

# 시작하기

Runway 2.0을 처음 사용하는 사용자를 위한 가이드입니다. 플랫폼 개요부터 로그인, 작업 환경 설정까지 단계별로 안내합니다.

---

## Runway 이해하기

-    **Runway 소개**

    ---

    AI·ML 워크로드를 일관된 방식으로 구성·배포·운영할 수 있는 플랫폼의 핵심 개념을 소개합니다.

     [자세히 보기](understanding-runway/overview.md)

-    **작업 단위와 계층 구조**

    ---

    워크스페이스와 프로젝트라는 두 가지 작업 단위를 기준으로 한 계층적 구조를 알아봅니다.

     [자세히 보기](understanding-runway/workspace-project.md)

-    **사용자 역할 및 접근 범위**

    ---

    워크스페이스와 프로젝트 단위로 부여되는 역할과 각 역할의 작업 범위를 파악합니다.

     [자세히 보기](understanding-runway/roles-and-permissions.md)

## 빠른 시작 가이드

-    **로그인 및 인증**

    ---

    Keycloak 기반의 통합 인증 시스템을 통해 플랫폼에 로그인하는 방법을 안내합니다.

     [자세히 보기](getting-ready/authentication.md)

-    **프로젝트 접근 및 참여**

    ---

    AI·ML 워크로드를 시작하기 위해 프로젝트 접근 권한을 요청하는 방법을 알아봅니다.

     [자세히 보기](getting-ready/project-access.md)

-    **첫 개발 환경 만들기**

    ---

    스토리지 볼륨을 생성하고 Code Server를 배포하여 브라우저에서 바로 코딩을 시작합니다.

     [자세히 보기](getting-ready/setup-quickstart.md)

-    **개발한 모델 서빙하기**

    ---

    학습한 머신러닝 모델을 프로덕션 환경에 배포하고 추론 엔드포인트를 생성합니다.

     [자세히 보기](getting-ready/model-serving-quickstart.md)

---

## 지원 브라우저

Runway는 최신 웹 표준을 준수하는 웹 기반 애플리케이션으로, 주요 최신 브라우저의 최신 안정화 버전을 공식 지원합니다.

지원 브라우저:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Apple Safari

> **Note**: 최신 안정화 버전 사용 권장
>
> Runway는 최신 웹 표준과 보안 정책을 기반으로 동작합니다.  
> 브라우저를 최신 버전으로 유지하면 UI 렌더링 정확성, 성능, 보안 및 기능 호환성을 안정적으로 보장할 수 있습니다.
